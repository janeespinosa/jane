¿ Que es Gitignore?
El archivo .gitignore se utiliza en un repositorio
git para ignorar los archivos y directorios que no son
necesarios para el proyecto. Git los ignorará una vez
que se hayan confirmado los cambios en el repositorio
remoto. Los tipos de archivos que se ignoran son
principalmente los archivos temporales y los
archivos que no deben tener versiones.




¿Qué son los archivos .keep y .gitkeep?
Se trata de archivos de marcador de posición que no tienen
contenido pero que sirven para indicar a Git que se debe realizar
un seguimiento de un directorio, incluso si está vacío.
Básicamente, envían una señal al sistema de control de versiones:
"Oye, este directorio existe y quiero que realices un seguimiento de él".
